# SPFx Pedido de Férias

Projeto SharePoint Framework (SPFx) para gerenciar pedidos de férias.

SPFx Pedido de Férias é um projeto desenvolvido utilizando o SharePoint Framework (SPFx) para facilitar o gerenciamento de pedidos de férias em uma organização. Este projeto tem como objetivo oferecer uma interface personalizada dentro do SharePoint, onde os colaboradores podem submeter suas solicitações de férias e acompanhar o status desses pedidos.

Funcionalidades principais:

Cadastro de Pedidos de Férias: Funcionários podem registrar seus pedidos de férias diretamente no SharePoint.

Acompanhamento do Status: Visualização do status de aprovação ou pendência dos pedidos.

Integração com SharePoint: Utiliza a plataforma SharePoint para armazenar e gerenciar dados de maneira eficiente.

Interface Personalizada: A interface do usuário é desenvolvida com a melhor experiência possível em mente, com uma integração fluida e responsiva.

Como usar:

Clone o repositório:
Use o comando git clone https://github.com/raq-boop/spfx-pedidos-ferias.git para clonar o projeto.

Instale as dependências:
Navegue até a pasta do projeto e execute npm install para instalar todas as dependências necessárias.

Configure e execute o projeto no SharePoint.

Tecnologias usadas:

SharePoint Framework (SPFx)

React (opcional, caso você utilize para interface)

Node.js e npm para gerenciamento de pacotes
